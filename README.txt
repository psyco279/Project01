Silahkan ikuti langkah-langkah ini untuk menggunakan program otomatis login instagram:
1. Silahkan install python interpreter, bisa anda cari di google, lalu mendowload dan menginstallnya
2. Install package python menggunakan pip di cmd atau terminal dari file requirements.txt
3. Silahkan install browser Firefox di karenakan program ini hanya berfungsi pada browser tersebut
4. Kemudian buka file credential_login.txt
5. isi sesuai username dan password akun instagram yang anda miliki
6. Akhir buka file login.py untuk menjalankan program, maka akan otomatis masuk ke akun instagram anda melalui browser

* catatan: 
hanya berfungsi untuk akun non autentikasi dua langkah, 
silahkan atur terlebih dahulu pada pengaturan jika autentikasi dua langkah masih aktif
